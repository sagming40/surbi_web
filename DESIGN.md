## 포함된 화면

| 화면 | 설명 |
|------|------|
| 로그인 | 구글 소셜 로그인. Step 1~3은 비로그인도 가능, Step 4는 로그인 필수 |
| Step 1 | 지역(시/구/동) + 업종 선택, 행정동 히트맵 미리보기 |
| Step 2 | 선택 지역 상권 분석 대시보드 (점수, 유동인구/경쟁업소 차트) |
| Step 3 | 지도 기반 건물 탐색, 마커 탭 시 BottomSheet (AR 모드는 제외됨) |
| Step 4 | AI 창업 점수 + SHAP 그래프 + LLM 보고서 생성 ⭐ 핵심 기능 |

## 참고 사항

- AR 모드는 교수님 피드백에 따라 제외되었으며, LLM 보고서 기능으로 대체되었습니다.
- 각 화면 하단에 팀원 이해를 돕기 위한 설명 텍스트가 함께 첨부되어 있습니다.
- 실제 코드 구현 시 Figma **Dev Mode**(우측 상단 디자인/개발 토글)에서 정확한 크기·색상 값을 확인할 수 있습니다.
